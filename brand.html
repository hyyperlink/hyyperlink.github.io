Name Usage

Consistency in naming is crucial for brand recognition.

Correct Usage:

  • Hyyperlink - For starting sentences and titles
  • hyyperlink - For general usage
  • HYYPERLINK - For all caps logos and enthusiastically shouting
  • HYY - Currency ticker

Incorrect Usage:

  • HyyperLink (should be Hyyperlink)
  • hyyperLink (should be hyyperlink)
  • hyper-link (There are two Y's in hyyperlink)
  • hyyper-link (Never break up the word)

Key Information Takeaways

  • Hyyperlink is an original Layer 1 protocol developed in the United States, built from the ground up to address the scalability limitations of traditional blockchain networks
  • Unlike traditional blockchains, Hyyperlink uses a Directed Acyclic Graph (DAG) architecture, enabling parallel transaction processing and eliminating block-based bottlenecks
  • Transactions are processed asynchronously and confirm immediately, with no waiting for block confirmations
  • Network performance improves with adoption - confirmation times decrease as transaction volume increases
  • Built-in support for tokens, NFTs, and DeFi features like swaps and liquidity pools without requiring Layer 2 solutions or external protocols
  • Fixed supply of 100 million HYY tokens with a sustainable 2% annual inflation rate
  • Deflationary mechanism: All transaction fees are permanently removed from circulation
  • Novel Key Discovery Mining: Miners search for rare Ed25519 keys independently, with no competition or specialized hardware requirements
  • Mining activity contributes to network throughput while being completely separate from transaction validation - the network operates independently of mining activity
  • Miners provide network inflation through key discovery, while all transaction fees are burned - creating a balanced economic model
  • Significantly lower resource requirements compared to traditional cryptocurrency mining - no specialized hardware arms race
